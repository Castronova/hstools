# HSTools

A humble collection of HydroShare tools written in Python.


## Getting Started

Connect to HydroShare.

```
./hs_init --help
```

Creating resources 

```
./hs_create --help
```

Getting resources

```
./hs_get --help
```

Adding files to an existing resource

```
./hs_add --help
```

Deleting a resource

```
./hs_delete
```
